== Description ==

This plugin allows you to add the official LOVEFiLM widget to the sidebar of your Wordpress blog. Monetise your blog by promoting the latest and most popular movies or games with LOVEFiLM's affiliate program.
For More Information Click here. (LINK to: http://www.lovefilm.com/partnership/widgets)

== Installation ==

1.	Install the plugin from within the Dashboard or upload the lovefilm folder and all its contents to the /wp-content/plugins/ directory
2.	Activate the plugin through the 'Plugins' menu in WordPress
3.	Drag and drop the widget onto your sidebar. Go to 'appearance' then 'widgets'
4.	Edit your preferences in the 'LOVEFiLM Widget' settings panel
